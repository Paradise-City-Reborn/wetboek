# 🚗 Rijontzeggingen & Inbeslagnames

### Artikel 7.1. - Overtredingen die een Strike geven

| *Overtreding* | *Strike* |
|---|---|
| Straatracen | +1 |
| Vluchten voor politie | +1 |
| Rijden onder zware invloed (alcohol/drugs) | +1 |
| Meerdere keren vluchten binnen korte periode | +1 |
| Opzettelijk inrijden op burger of agent | +1 |
| Overval (met uitzondering van woning- en winkeloverval) | +1 |
| Voertuig gebruikt bij schietpartij of gewelddadig incident | +1 |
| Strike kan ook naar eigen inzicht door command of hoger worden toegekend | +1 |

### Artikel 7.2. - Gevolgen van Strikes

| *Aantal strikes* | *Actie* |
|---|---|
| 1 strike | Waarschuwing |
| 2 strikes | Laatste waarschuwing |
| 3 strikes | Voertuig wordt permanent ingenomen en later geveild |

### Artikel 7.3. - Samenvatting Voertuigmaatregelen

| *Overtreding* | *Consequentie* |
|---|---|
| 3 strikes | Voertuig permanent ingenomen + later geveild |
| Palm Coin voertuigen | 1 week ingenomen + waarde terugbetalen |
| Echt geld voertuigen | 1 week ingenomen |
| 1 Palm Coin | 30k |
| €1 | 25k |
